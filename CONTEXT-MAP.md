# Context Map

## Contexts

- [Planning](./product/CONTEXT.md) — defines Planner's platform-neutral Google Calendar language
- [Web Experience](./web/CONTEXT.md) — presents the planning experience through the web delivery stack

## Relationships

- **Web Experience → Planning**: Web Experience uses Planning's account, source-calendar, event, and local-date language while owning its own presentation, refresh, and offline-display concepts.
